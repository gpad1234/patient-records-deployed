================================================================================
                         TROUBLESHOOTING
================================================================================

Service won't start?
  1. Check port is available: lsof -i :<port>
  2. Kill if needed: ./scripts/kill-port.sh <port>
  3. Check logs in terminal
  4. See START_HERE_NO_DOCKER.md troubleshooting section

Database connection failed?
  1. Check PostgreSQL is running: ./scripts/start-postgres.sh
  2. Initialize database: ./scripts/init-database.sh
  3. Test connection: psql -h localhost -U patient_user -d patient_records

Service health check fails?
  1. Ensure all services are running
  2. Check logs in each service terminal
  3. Wait 30 seconds for services to fully start
  4. Run ./scripts/test-services.sh again

Build takes too long?
  1. First Java build: 2-3 minutes (normal, downloading dependencies)
  2. Subsequent builds: 30-60 seconds
  3. This is expected behavior
